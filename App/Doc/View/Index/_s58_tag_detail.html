58.话题详情

接口地址：测试url
请求方式：POST
service 文本 是 tag_detail
tag_id 文本 是 tag id
my_user_id 文本 否 当前用户的user_id
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • count 话题对应的视频总数
  • tag 话题详情
    • id 话题id
    • is_recommend 是否推荐
    • name 话题名
    • photo_url 封面图
    • bg_photo_url 背景图
    • intro 简介
    • is_hot 是否热门
    • view_sum 浏览数
    • collect_sum 分享数
    • pub_times 关联次数
  • video 视频列表, 数组, 包含多个【视频元数据】
    • 视频元数据 同接口 5.发现
