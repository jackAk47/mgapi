44.二级评论列表

接口地址：测试url
请求方式：POST
service 文本 是 comment_list
my_user_id 文本 否 当前用户的user_id
type 文本 是 类型 comment
relation_id 文本 是 一级评论的 comment_id
current_page 文本 否 分页参数，页码从1开始，默认值是1。
page_size 文本 否 分页参数，每页显示的视频数量，默认值是20。

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • top 数组, 一级评论相关信息
    • liked_sum 赞过人的数量
    • liked 数组, 赞过的人的头像URL
    • detail 一级评论相关信息, 数组
      • id 评论id
      • user_id 用户ID
      • topid 项目id
      • type 评论类型
      • pub_time 评论时间
      • text 评论内容
      • liked 赞过的次数
      • is_liked 当前用户是否点赞
      • user_name 评论用户名
      • user_avatar 评论用户头像
  • comment 数组, 包含多个【二级评论元数据】
    • 二级评论元数据
      • type comment(类型)
      • is_liked 当前用户(my_user_id)是否点赞过
      • id 二级评论id
      • pid 一级评论id
      • pub_time 发布时间
      • user_id 发布创客的user_id
      • user_name 发布创客的名称
      • user_avatar 发布创客的头像
      • text 评论内容
      • liked 点赞数
