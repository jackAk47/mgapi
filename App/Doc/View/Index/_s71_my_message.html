71.我的消息

接口地址：测试url
请求方式：POST
service 文本 是 my_message
my_user_id 文本 是 当前用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 我的消息
    • message_sum 未读消息总数
    • push 系统推送数据
      • message_sum 未读消息总数
      • last_message_time 最近一条消息的时间
      • last_message_desc 最近一条消息的描述
    • comment 评论通知数据
      • message_sum 未读消息总数
      • last_message_time 最近一条消息的时间
      • last_message_desc 最近一条消息的描述
    • like 点赞通知数据
      • message_sum 未读消息总数
      • last_message_time 最近一条消息的时间
      • last_message_desc 最近一条消息的描述
    • follow 关注通知数据
      • message_sum 未读消息总数
      • last_message_time 最近一条消息的时间
      • last_message_desc 最近一条消息的描述
