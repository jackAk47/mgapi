74.评论通知列表

接口地址：测试url
请求方式：POST
service 文本 是 comment_message_list
my_user_id 文本 是 当前用户的user_id
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • count 消息总数
  • data 消息列表，数组，包含多个【消息元数据】
    • 消息元数据
      • id评论记录id
      • userid去评论的用户ID
      • topid评论的对应项目|视频|评论的ID
      • addtime评论添加时间
      • type评论的类型：项目|视频|评论
      • photo_url评论的对应项目|视频|评论的图片URL
      • user_name评论用户名
      • user_avatar评论用户头像
