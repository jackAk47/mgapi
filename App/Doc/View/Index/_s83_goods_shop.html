83.商城首页

接口地址：测试url
请求方式：POST
service 文本 是 goods_shop
my_user_id 文本 否 用户id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 商品首页
    • collect_sum 用户商品收藏总数
    • banner 轮播图，数组，包含多个轮播图URL
    • branch 商品分类图标，数组，包含多个【分类图标元数据】
      • 分类图标元数据
        • name分类名
        • image图标URL
    • theme 主题商品列表，数组，包含多个【主题元数据】
      • 主题元数据 商品列表，数组，包含多个【商品元数据】
        • 商品元数据
          • 商品元数据 同接口 8.1.商品列表
