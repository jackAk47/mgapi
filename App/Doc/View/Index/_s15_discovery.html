5.发现

接口地址：测试url
请求方式：POST
service 文本 是 discovery
my_user_id 文本 否 当前用户的user_id
branch 文本 否 分类名称 recommend|super_equipment|cosmic_sky等, 默认recommend
sort_type 文本 否 排序类型 1|2|3 1最新,2最热,3关注, 默认1
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • banner2 数组，包含多个【banner元数据】
    • banner元数据
      • id轮播图id
      • type轮播类型
      • value轮播图类型的id或者url
      • photo_url轮播图的链接
      • is_recommend1:默认推荐/上架
      • recommend_time推荐/上架时间
      • update_admin更新人
  • user 当前用户(my_user_id)的关注创客, 数组, 包含多个user_id
  • branch 视频项目的分类, 数组，包含多个分类
  • data 发现列表, 数组, 包含多个【视频元数据】和【众筹元数据】
    • 视频元数据
      • type video(类型)
      • is_liked 当前用户(my_user_id)是否点赞过
      • id 视频id
      • pub_time 发布时间
      • user_id 发布创客的user_id
      • user_name 发布创客的名称
      • user_avatar 发布创客的头像
      • video_url 视频url
      • title 视频标题
      • photo 视频封面图
      • introduction 视频简介
      • view_times 浏览量
      • comments 话题数
      • liked 点赞数
      • collection 分享数
      • hot 热度
    • 众筹元数据
      • type project(类型)
      • is_liked 当前用户(my_user_id)是否点赞过
      • id 众筹id
      • pub_time 发布时间
      • user_id 发布创客的user_id
      • user_name 发布创客的名称
      • user_avatar 发布创客的头像
      • video_url
      • p_photo
      • p_summary
      • p_name
      • p_current_money
      • progress
      • p_days
      • view_times 浏览量
      • comments 话题数
      • liked 点赞数
      • collection 分享数
      • hot 热度
