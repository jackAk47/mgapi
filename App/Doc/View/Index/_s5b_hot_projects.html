5b.热门众筹

接口地址：测试url
请求方式：POST
service 文本 是 hot_projects
my_user_id 文本 否 当前用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 众筹列表, 数组, 包含多个【众筹元数据】
    • total 累计值, 数组, 包含累计筹资和累计支持数
      • money 累计筹资
      • support 累计支持数
    • bottom 底部图片, 数组, 包含图片URL和跳转URL
      • photo_url 图片URL
      • jump_url 跳转URL
    • branch_name 分类名, 数组, 包含所有的分类名称
    • branch_image 分类图片, 数组, 包含所有的分类对应的图片url（ios，ndr）
    • 热门元数据 同接口 5.发现
    • 智能硬件元数据 同接口 5.发现
    • 虚拟现实元数据 同接口 5.发现
    • 机器人元数据 同接口 5.发现
    • 设计元数据 同接口 5.发现
