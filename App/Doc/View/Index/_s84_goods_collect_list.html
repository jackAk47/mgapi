84.收藏商品列表

接口地址：测试url
请求方式：POST
service 文本 是 goods_collect_list
my_user_id 文本 否 用户id
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • count 商品总数
  • data 商品列表，数组，包含多个【商品元数据】
    • 商品元数据
      • id商品id
      • name商品名
      • photo商品图片
      • price商品价格
      • tb_url淘宝商品跳转链接
      • is_recommend是否上架
      • type类型
