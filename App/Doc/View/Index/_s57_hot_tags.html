57.热门话题

接口地址：测试url
请求方式：POST
service 文本 是 hot_tags
my_user_id 文本 否 当前用户的user_id
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • count 话题总数
  • data 话题列表, 数组, 包含多个【标签元数据】
    • 标签元数据
      • id 标签id
      • name 话题名
      • intro 简介
      • photo_url 封面图
      • view_sum 浏览数
