6.视频详情

接口地址：测试url
请求方式：POST
service 文本 是 video_detail
my_user_id 文本 否 当前用户的user_id
video_id 文本 是 视频id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • video 视频详情
    • type video(类型)
    • is_liked 当前用户(my_user_id)是否点赞过
    • id video_id
    • pub_time 发布时间
    • user_id 发布创客的user_id
    • user_name 发布创客的名称
    • user_avatar 发布创客的头像
    • video_url 视频url
    • title 视频标题
    • photo 视频封面图
    • introduction 视频简介
    • view_times 浏览量
    • comments 评论数
    • liked 点赞数
    • collection 分享数
    • hot 热度
    • tags 视频标签, 数组
    • content 视频内容
    • fan_sum 发布创客的粉丝数
    • fan_list 发布创客的最新关注的6个粉丝
    • is_followed 当前用户(my_user_id)是否关注发布创客
  • comment 视频评论, 数组, 包含多个【一级评论元数据】
    • 一级评论元数据 同接口 42.一级评论列表
  • project 正在众筹的项目, 数组, 包含多个【众筹元数据】
    • 众筹元数据 同接口 5.发现
  • you_may_like 猜你喜欢, 数组, 包含多个【视频元数据】
    • 视频元数据 同接口 5.发现
