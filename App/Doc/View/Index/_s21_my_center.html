21.我的个人中心

接口地址：测试url
请求方式：POST
service 文本 是 my_center
my_user_id 文本 是 当前用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 当前用户的相关信息
    • user_id 当前用户ID
    • user_name 用户名
    • user_avatar 用户头像
    • province 用户地区 省市区
    • gender 用户性别
    • fans 用户粉丝数
    • follow 用户关注数
    • introduction用户个人介绍
    • creative 用户创造力
    • videos 用户视频数
    • projects 用户众筹数
    • orders 用户订单数
    • ugc_draft_sum 草稿箱数量
    • collect_sum 收藏商品数量
  • video 视频列表, 数组, 包含多个【视频元数据】
    • 视频元数据 同接口 5.发现
  • project 项目列表, 数组, 包含多个【众筹元数据】
    • 众筹元数据 同接口 5.发现
