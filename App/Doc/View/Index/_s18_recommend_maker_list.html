8.推荐创客列表

接口地址：测试url
请求方式：POST
service 文本 是 recommend_maker_list
my_user_id 文本 否 当前用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 推荐创客列表, 数组，包含多个【创客元数据】
    • 创客元数据
      • user_id 用户ID
      • user_name 用户名
      • user_avatar 用户头像
      • introduction用户个人介绍
