5d.赞过的人

接口地址：测试url
请求方式：POST
service 文本 是 liked_man
type 文本 是 类型 视频/众筹/话题 3选1 video|project|comment
relation_id 文本 是 video|project|comment 的id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 赞过的用户列表，数组，包含多个【用户信息元数据】
    • 用户信息元数据
      • user_id用户ID
      • add_time点赞时间
      • user_name用户名
      • user_avatar用户头像
