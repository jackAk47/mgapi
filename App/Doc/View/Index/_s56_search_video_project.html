56.App搜索

接口地址：测试url
请求方式：POST
service 文本 是 search_video_project
keyword 文本 否 搜索关键词
type 文本 是 搜索类型：video project goods
current_page 文本 否 当前页数, 默认1
page_size 文本 否 每页的视频数量, 默认20

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • count 众筹或视频总数
  • data 数组,众筹 视频 商品列表 包含多个【众筹 视频 商品元数据】
    • 众筹或视频元数据
      • 众筹或视频元数据 同接口 5.发现
    • 商品元数据
      • 商品元数据 同接口 8.1.商品列表
