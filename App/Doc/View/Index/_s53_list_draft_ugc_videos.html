53.ugc视频草稿列表

接口地址：测试url
请求方式：POST
service 文本 是 list_draft_ugc_videos
my_user_id 文本 是 当前用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data 视频列表, 数组, 包含多个【视频元数据】
    • UGC视频元数据
      • id 视频id
      • title 视频标题
      • photo_url2 图片URL
      • video_url2 图片URL
      • update_time 更新时间
