31.TA的个人中心

接口地址：测试url
请求方式：POST
service 文本 是 maker_center
my_user_id 文本 否 当前用户的user_id
other_user_id 文本 是 其他的用户的user_id

返回JSON值说明：

  • res 状态，true/false 请求成功或者失败的判别
  • msg 消息
  • code 特定情况下返回，001签名错误，002必须参数不完整(参数为空)
  • data TA的用户相关信息
    • user_name 用户名
    • user_avatar 用户头像
    • fans 用户粉丝数
    • introduction用户个人介绍
    • creative 用户创造力
    • is_followed 当前用户对TA是否关注 true/false
  • video 视频列表, 数组, 包含多个【视频元数据】
    • 视频元数据 同接口 5.发现
